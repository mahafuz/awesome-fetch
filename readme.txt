== Description ==

A simple plugin that retrieves data from a remote endpoint,
and makes it data accessible/retrievable from an AJAX endpoint on the WordPress site.
The data will be also displayed via a shortcode and on an admin WordPress dashboard page.
It has WP CLI command also for the force update of data.

A few notes about the sections above:

*   ShortCode to display the data on the frontend `[awf_table]`
*   Cli Command to force update the data `wp awesome-fetch`


== Installation ==

1. Upload `awesome-fetchp` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[awf_table]` in your pages to display the data.
